RU Русский
Светильник оснащён регулировкой яркости, таймером отключения и функцией сохранения настроек, которые используются как значения по умолчанию после повторного подключения к источнику питания.
1. Включение
Для включения кратко нажмите кнопку.
Когда свет включён, яркость можно изменять короткими нажатиями на кнопку.
2. Сохранение яркости
Чтобы сохранить выбранную яркость как значение по умолчанию, нажмите и удерживайте кнопку в течение 1 секунды (сработает соответствующая анимация).
Это необходимо сделать не позднее чем через 2 секунды после изменения яркости, иначе светильник просто выключится.
3. Выключение
Чтобы выключить свет, нажмите и удерживайте кнопку в течение 1 секунды.
Если перед этим изменялась яркость, должно пройти не менее 2 секунд, иначе будет сохранена яркость, а светильник останется включённым.
4. Таймер
Для использования таймера светильник должен быть выключен (не излучать свет), но подключён к источнику питания.
В этом состоянии нажмите и удерживайте кнопку в течение 1 секунды — будет активирован режим настройки таймера.
Затем короткими нажатиями кнопки увеличивайте время.
Отображение осуществляется с помощью двух LED-колец и центрального светодиода:
•	Центральный светодиод вместе с кольцами образует опорную линию
•	Малое кольцо показывает часы (1 светодиод = 1 час)
•	Большее кольцо показывает минуты (1 светодиод = 5 минут)
Важно:
•	Сначала время увеличивается с шагом 5 минут
•	После превышения 30 минут шаг увеличивается до 10 минут
•	Максимальное время — 120 минут
После выбора нужного значения:
•	Нажмите и удерживайте кнопку 1 секунду для сохранения
или
•	Подождите 2 секунды — светильник автоматически включится и начнётся обратный отсчёт
По окончании таймера свет автоматически выключится.
Примечание:
Функция сохранения яркости и таймера упрощает использование устройства.
Сохранённые значения применяются только после повторного подключения к питанию и используются как начальные настройки.
Если питание не отключалось, используются последние активные значения.
Отладочная функция (логирование):
Функция логирования доступна при подключении к микроконтроллеру через RS232.
Для активации: удерживайте кнопку, подключите устройство к питанию и продолжайте удерживать кнопку ещё около 2 секунд.
 
DE Deutsch:
Die Leuchte verfügt über eine Helligkeitsregelung, einen Abschalttimer sowie eine Speicherfunktion, mit der die eingestellten Parameter nach dem erneuten Anschließen an die Stromversorgung als Standardwerte verwendet werden.
1. Einschalten
Zum Einschalten die Taste kurz drücken. Wenn das Licht eingeschaltet ist, kann die Helligkeit durch kurze Tastendrücke angepasst werden.
2. Helligkeit speichern
Um die gewählte Helligkeit als Standard zu speichern, die Taste für 1 Sekunde gedrückt halten (eine entsprechende Animation bestätigt die Speicherung).
Dies muss spätestens innerhalb von 2 Sekunden nach der letzten Helligkeitsänderung erfolgen, sonst schaltet sich die Leuchte einfach aus.
3. Ausschalten
Zum Ausschalten die Taste 1 Sekunde gedrückt halten.
Falls zuvor die Helligkeit geändert wurde, müssen mindestens 2 Sekunden vergangen sein, da sonst die Helligkeit gespeichert wird und die Leuchte eingeschaltet bleibt.
4. Timer-Funktion
Um den Timer zu nutzen, muss die Leuchte ausgeschaltet sein (kein Licht), aber an die Stromversorgung angeschlossen bleiben.
In diesem Zustand die Taste 1 Sekunde gedrückt halten, um den Timer-Modus zu aktivieren.
Danach kann die Zeit durch kurze Tastendrücke erhöht werden.
Die Anzeige erfolgt über zwei LED-Ringe und die zentrale LED:
•	Die zentrale LED bildet zusammen mit den beiden kleineren Ringen eine Referenzlinie
•	Der kleinere Ring zeigt Stunden an (1 LED = 1 Stunde)
•	Der größere Ring zeigt Minuten an (1 LED = 5 Minuten)
Hinweis:
•	Zunächst wird die Zeit in 5-Minuten-Schritten erhöht
•	Nach Überschreiten von 30 Minuten erfolgt die Erhöhung in 10-Minuten-Schritten
•	Die maximale Zeit beträgt 120 Minuten
Nach Auswahl der gewünschten Zeit:
•	Taste 1 Sekunde gedrückt halten, um zu speichern
oder
•	2 Sekunden warten – die Leuchte schaltet sich automatisch ein und der Countdown startet
Nach Ablauf des Timers schaltet sich das Licht aus.
Hinweis:
Die Speicherfunktion für Helligkeit und Timer erleichtert die Nutzung.
Gespeicherte Werte werden nur nach dem erneuten Anschluss an die Stromversorgung als Startwerte verwendet.
Wird die Leuchte nicht vom Strom getrennt, bleiben die zuletzt verwendeten Einstellungen aktiv.
Debug-Funktion (Logging):
Die Logging-Funktion kann über eine RS232-Verbindung zum Mikrocontroller genutzt werden.
Zum Aktivieren: Taste gedrückt halten, Gerät an die Stromversorgung anschließen und die Taste noch etwa 2 Sekunden weiter gedrückt halten.
 
EN English:
The lamp features brightness control, a shutdown timer, and a memory function that saves the selected settings to be used as default after reconnecting to the power supply.
1. Power on
Press the button briefly to turn the light on.
When the light is on, brightness can be adjusted with short button presses.
2. Save brightness
To save the selected brightness as the default, press and hold the button for 1 second (an animation will confirm the action).
This must be done within 2 seconds after adjusting the brightness, otherwise the lamp will simply turn off.
3. Power off
To turn the light off, press and hold the button for 1 second.
If the brightness was adjusted beforehand, at least 2 seconds must pass; otherwise, the brightness will be saved and the light will remain on.
4. Timer function
To use the timer, the lamp must be switched off (no light) but still connected to power.
In this state, press and hold the button for 1 second to enter timer setup mode.
Then use short button presses to increase the time.
The time is displayed using two LED rings and a central LED:
•	The central LED together with the rings forms a reference line
•	The smaller ring shows hours (1 LED = 1 hour)
•	The larger ring shows minutes (1 LED = 5 minutes)
Note:
•	Time increases in 5-minute steps initially
•	After exceeding 30 minutes, it increases in 10-minute steps
•	Maximum time is 120 minutes
After selecting the desired time:
•	Press and hold the button for 1 second to save
or
•	Wait 2 seconds – the light will turn on automatically and the countdown will start
When the timer reaches zero, the light turns off.
Note:
The memory function for brightness and timer improves usability.
Saved values are applied only after reconnecting the lamp to power and will be used as initial settings.
If the lamp remains powered, it will continue using the last active settings.
Debug function (logging):
A logging function is available via RS232 connection to the microcontroller.
To activate it: hold the button, connect the device to power, and keep holding the button for about 2 seconds.
